jsp와 javaScript의 차이점
==> 웹프로그래밍 언어

jsp: 서버사이드. 서버에서 실행해서 브라우저에 전송
javaScript: 클라이언트(웹브라우저) 사이드. 브라우저에서 실행

ajax 문서의 일부분의 갱신을 위해 필요한 데이터만 응답

CSV(comma-separated values) 
json 자바스크립트의 객체
xml

json의 형태
{
	dno:100,
	dname:"기획부",
	dloc:"서울"
}


[{
	dno:100,
	dname:"기획부",
	dloc:"서울"
},
{
	dno:100,
	dname:"기획부",
	dloc:"서울"
},
..
] 여러개를 표현시 []안에 포함.
